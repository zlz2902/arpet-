# 3D 宠物模型 — 设计与接入说明

> 产品向项目（非毕设）。当前不做数据库；模型以 **GLB** 形式内置在 App 中，摄像头画面上叠加显示。

## 当前已内置模型

| ID | 显示名 | 文件 | 说明 |
|----|--------|------|------|
| `duck` | 小黄鸭 | `www/models/duck.glb` | Khronos 官方示例 Duck |
| `fox` | 小狐狸 | `www/models/fox.glb` | Khronos Fox |
| `brainstem` | 星云精灵 | `www/models/brainstem.glb` | 占位用抽象造型，可替换 |

主页四个卡片通过 `pets.json` → `homepageMap` 映射到上述模型（**请使用自有或已授权模型替换宝可梦形象**）。

## 目录结构

```
app/src/main/assets/www/
├── pets.json          # 宠物清单：路径、缩放、位置、主页映射
├── ar_mobile.html     # AR 页：相机 + A-Frame 透明 3D 层
├── models/            # 所有 .glb 放这里
│   ├── duck.glb
│   ├── fox.glb
│   └── brainstem.glb
└── js/aframe.min.js
```

改完 Android 资源后，同步到 iOS：

```powershell
Copy-Item app\src\main\assets\www\* ios\ARPet\Resources\www\ -Recurse -Force
```

## 如何新增一只宠物

1. 将 `.glb` 放入 `app/src/main/assets/www/models/`，例如 `my_pet.glb`。
2. 编辑 `pets.json`，在 `pets` 数组增加一项：

```json
{
  "id": "my_pet",
  "name": "我的宠物",
  "model": "models/my_pet.glb",
  "scale": "1 1 1",
  "position": "0 0 -1.2",
  "rotation": "0 0 0"
}
```

3. 在 AR 页底部切换条会自动出现新按钮（读取 `pets.json`）。
4. 若要从主页某张卡片进入，在 `homepageMap` 里增加映射，并改 `index.html` 的 `appPetMap`（与 `homepageMap` 保持一致）。

### 缩放与位置怎么调

- 模型在镜头前 **约 1～2 米** 处：`position` 的 Z 一般为 **-0.8 ~ -2.0**（越负越远）。
- 不同 GLB 尺寸差很多：Fox 常需 `scale` **0.01** 量级，Duck 约 **0.4**。
- 在真机上改 `pets.json` → 重新 Run App → 进 AR 页底部切换对比。

## 模型从哪来（合法来源）

| 来源 | 说明 |
|------|------|
| [Khronos glTF Sample Models](https://github.com/KhronosGroup/glTF-Sample-Models) | 学习/占位，许可见仓库 |
| [免费3D模型推荐.md](./免费3D模型推荐.md) | **已整理**：Quaternius / Poly Pizza（CC0）等可直接替换链接 |
| 自建 Blender → 导出 GLB | 推荐，完全自有 |
| Sketchfab / TurboSquid 等 | 必须看清 **商用许可** |
| 宝可梦官方形象 | **勿未授权使用**；主页文案可保留「小精灵」概念，模型需替换 |

### Blender 导出建议

- 格式：**glTF 2.0 Binary (.glb)**
- 面数：手机建议 **&lt; 50k 三角面** 单宠
- 贴图：尽量 **一张** 主贴图，尺寸 **1024×1024** 或更小
- 原点：脚底或身体中心，朝向 **-Z** 朝相机（与 A-Frame 默认一致）

## 技术说明（AR 页）

- **不用 AR.js**（荣耀 WebView 易卡死），采用 **`<video>` 相机 + 透明 A-Frame 场景**。
- 不是平面锚定 AR：宠物固定在镜头前方，随设备移动有「跟屏」感，适合当前阶段验证模型表现。
- 后续若要做「点地面放置」，需华为 AREngine / ARCore 等原生方案或稳定 WebXR。

## 常见问题

| 现象 | 处理 |
|------|------|
| 白屏只有 × | 多为 A-Frame 盖住相机；已用透明场景，仍异常则看是否加载了 `aframe.min.js` |
| 模型巨大/看不见 | 改 `pets.json` 的 `scale` / `position` |
| 模型不显示 | 检查 `models/xxx.glb` 是否在 assets；路径是否与 `pets.json` 一致 |
| 加载很慢 | 单个 GLB 建议 **&lt; 5MB**；压缩贴图、减面 |
