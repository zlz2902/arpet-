## 下载步骤（Poly Pizza）

1. 打开上表链接 → 点 **Download**（或 **GLB**）。  
2. 若只有 GLTF，用 [glTF Viewer](https://gltf-viewer.donmccurdy.com/) 或 Blender 另存为 **`.glb`**。  
3. 复制到：`app/src/main/assets/www/models/`，例如 `fox_quaternius.glb`。  
4. 编辑 `pets.json`：

```json
{
  "id": "fox",
  "name": "小狐狸",
  "model": "models/fox_quaternius.glb",
  "scale": "0.8 0.8 0.8",
  "position": "0 -0.2 -1.4",
  "rotation": "0 25 0"
}
```

5. 真机打开 AR 页，在底部切换；**scale / position 每个模型都要单独调**（Quaternius 模型比 Khronos 示例大很多时，scale 常用 `0.5～2`）。

## 许可与上架提醒

- **不要用**未授权的皮卡丘、杰尼龟等官方宝可梦模型。  
- **CC0**：一般可商用、可修改、可不署名（以页面为准）。  
- **CC-BY**：可商用，但需 **署名** 作者。  
- **Standard / 禁止商用**：不要放进正式 App。  
- 单模型建议 **&lt; 5MB**、**&lt; 5 万面**，否则 WebView 加载慢。
